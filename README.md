# tinybox

window manager based started from tinywl;

'box' ~ tries to emulate the blackbox, fluxbox, openbox family of wm

## features

* styles
* workspaces
* menu
* keybinding
* three/four fingers drag/swipe
* shade/slit
* mini titlebars
* auto arrange/resize windows
